## What I do
- Execute common git operations (add, commit, status, diff, log, branch)
- Analyze commit patterns and history
- Help with merge conflict resolution
- Manage git stash operations

## Common Operations
- `git status` - Check repository state
- `git add .` - Stage all changes
- `git commit -m "message"` - Commit with message
- `git diff` - View changes
- `git log --oneline -10` - Recent commits
- `git branch -a` - List all branches

## Analysis Features
- Commit message analysis
- Change frequency tracking
- Contributor statistics
- Branch relationship mapping

## When to use me
Use this for any git operations. Essential for Guardian (commits) and Librarian (context tracking) agents.
