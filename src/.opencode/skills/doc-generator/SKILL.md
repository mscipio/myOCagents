## What I do
- Generate README.md from project structure and package.json
- Create API documentation from code comments (JSDoc, docstrings)
- Build documentation tables for CLI tools
- Extract and organize code examples
- Maintain documentation structure consistency

## Output Formats
- **README.md:** Project overview, installation, usage
- **API Docs:** Function signatures, parameters, return types
- **CLI Docs:** Command descriptions, flags, examples
- **CHANGELOG.md:** Auto-generate from git commits

## When to use me
Use this when starting a new project or updating documentation. Essential for TechnicalWriter agent.
