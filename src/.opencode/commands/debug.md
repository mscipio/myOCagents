# Command: debug
## Role: Systematic Debugging
## Version: 1.0.0

### I. PURPOSE
Debug issues using systematic methodology. Find root cause, not symptoms.

### II. EXECUTION LOGIC
1. **HYPOTHESIZE:** Generate specific theories about bug cause
2. **INSTRUMENT:** Add targeted logging to test each hypothesis
3. **REPRODUCE:** Trigger the bug while capturing logs
4. **ANALYZE:** Examine evidence to identify root cause
5. **FIX:** Apply fix and verify

### III. SKILLS
- Use `debug-workflow` skill

### IV. RULES
- One hypothesis at a time
- Evidence-based reasoning
- Fix root cause, not symptoms
- Document learnings

### V. EXPECTED OUTPUT
Root cause identified, fix applied, tests passing
